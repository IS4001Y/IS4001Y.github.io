Networking Events at Western

Trying to find a job can be stressful, absolutely. Luckily, Western actually offers a number of resources to make it easier for you to achieve this.

For one, Western holds a number of job-related/networking events throughout the year. You can find information on these events here (Login -> Event Calendars -> All Events). On this page, you’ll be able to see a list of all events Western is offering. They can range from sessions on how to write a resume, to networking events with specific companies. All you have to do is register for the event and sign up! These can be very useful, and are actually not well known about.
